INSTALL INSTRUCTIONS:

Place this pk3 in your jDRP.box folder e.g:

C:\Program Files\Doomsday\Snowberry\Addons\jDRP.box

By installing this module into the root of your jdrp.box folder you
are declaring that you want this module to be enabled by default.

If however you want this module to be disabled by default,
install it into your jDRP.box\Extra folder e.g:

C:\Program Files\Doomsday\Snowberry\Addons\jDRP.box\Extra

Enjoy!
