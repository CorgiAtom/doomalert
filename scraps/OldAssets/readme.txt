INSTALL INSTRUCTIONS:

Place the jdrp.box folder in C:\Program Files\Doomsday\Snowberry\Addons    (if this is where you installed Doomsday)
Run Doomsday and check jDoom Resource Pack in the addons tab for either Doom, Doom 2, TNT or Plutonia.


Other installation methods:

1. Place jdrp.box folder into a custom addon folder you assigned with snowberry.

2. Unzip jdrp.box to Desktop, run snowberry go to the addon tab, click the + sign (add an addon) and click the ... button. Click the desktop folder and click ok. Select jdrp.box and install.
